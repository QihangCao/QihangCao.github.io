# Qihang Cao Homepage

Static personal homepage inspired by compact academic profile layouts.

## Edit

- `index.html`: content, links, publications, news, honors, services.
- `styles.css`: visual design and responsive layout.
- `script.js`: small runtime behavior.

## Publish with GitHub Pages

1. Create a repository named `QihangCao.github.io` under your GitHub account.
2. Upload these files to the repository root.
3. GitHub Pages will serve the site at `https://QihangCao.github.io/`.

If you already have a Pages repository, replace its root files with this version.
